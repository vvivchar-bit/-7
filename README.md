## Короткий опис

У межах практичного заняття було реалізовано:

- базовий клас `Chislo`, який містить змінну типу `long`;
- віртуальну функцію `factorial()` для обчислення факторіала числа;
- похідний клас `Matrix`, який успадковує `Chislo` та містить одномірний масив із 100 цілих додатних чисел;
- обчислення факторіалів елементів масиву;
- використання вказівника на базовий клас для виклику віртуальної функції.
